Get a Runlog API key

Runlog is the external-dependency knowledge layer for your agent — a cross-org registry of verified knowledge about public APIs, frameworks, and protocols. Registering gives you an API key you'll paste into your MCP client's config.

Submitting your email is processed by the data controller (see Impressum) for the sole purpose of issuing your API key and binding it to your account. No use beyond issuance, no third-party sharing beyond the email-delivery service. Full disclosure: Privacy Notice.

Mit dem Absenden wird Deine E-Mail-Adresse vom Verantwortlichen (siehe Impressum) ausschließlich zur Ausstellung Deines API-Keys und zur Bindung an Dein Konto verarbeitet. Keine darüber hinausgehende Nutzung, keine Weitergabe außer an den E-Mail-Dienstleister. Vollständige Auskunft: Datenschutzerklärung.

Once you have a key, head to Quickstart for the 4-step setup (register → wire MCP → drop in the skill → use Claude normally) or browse Agents for the per-vendor install paths covering Claude Code, Cursor, Cline, Continue, Windsurf, Aider, VS Code Copilot, JetBrains AI, and Zed.

Runlog's verifier, schema, vocabularies, skills, and website are open source under runlog-org; the MCP server and architectural docs are not public for now.
